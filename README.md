# Fluoresynths

Pronunciation: _fluorescence_

A flourescent-on-black color theme for Visual Studio (VS) Code based on SynthWave'84 and FluoroMachine.

## Installation

```bash
code --install-extension fluoresynths-0.0.1.vsix
```

## Build from Source

```bash
npm install -g @vscode/vsce
vsce package
```
